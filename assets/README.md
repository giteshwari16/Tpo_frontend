# IMRD College Assets

This directory contains branding assets for the IMRD TPO Portal.

## Required Images

### 1. College Logo (`college-logo.png`)
- **Format**: PNG with transparent background
- **Size**: 80px x 80px (optimal)
- **Style**: Circular logo preferred
- **Location**: Replace the placeholder file with your actual college logo

### 2. College Image (`college-image.png`)
- **Format**: PNG or JPG
- **Size**: 800px x 400px (optimal)
- **Style**: College building, campus, or official image
- **Location**: Replace the placeholder file with your actual college image

## How to Update

1. **College Logo**: 
   - Replace `college-logo.png` with your college logo
   - Ensure the logo has a transparent background for best results
   - The logo will appear in:
     - Login page
     - Sidebar navigation
     - Admin panel favicon

2. **College Image**:
   - Replace `college-image.png` with your college building or campus image
   - This image appears in the main dashboard header
   - Recommended aspect ratio: 2:1 (width:height)

## Image Guidelines

- Use high-quality images for professional appearance
- Ensure logos are readable at small sizes
- College images should represent the institution well
- File sizes should be optimized for web loading

## Current Status

The portal is configured and ready to display the IMRD branding once the actual images are added to this directory.
